Easily create custom order statuses for WooCommerce with personalized email notifications.

== Description ==

AS Custom Order Status for WooCommerce is a powerful plugin that allows store administrators to create unlimited custom order statuses with personalized email notifications. Perfect for businesses that need to track specific stages in their order fulfillment process.

**Key Features:**

* **Unlimited Custom Statuses**: Create as many custom order statuses as your business needs
* **Personalized Email Notifications**: Configure unique email subject, heading, and content for each status
* **Easy Management**: Intuitive admin interface for adding, editing, and deleting custom statuses
* **Email Preview**: Preview email content before saving to ensure it looks perfect
* **Shortcode Support**: Use shortcodes like {order_number}, {customer_name}, and {order_date} in your emails
* **Custom Colors**: Set custom colors for each status to make them easily recognizable in the admin area
* **WooCommerce Integration**: Seamlessly integrates with WooCommerce order management system

**Perfect For:**

* Shoe repair services (like your example: Shoes Received, Shoes Ready, etc.)
* Custom manufacturing businesses
* Order fulfillment centers
* Any business with a multi-stage order process

**How It Works:**

1. Go to WooCommerce > Custom Order Statuses
2. Add a new custom status with your preferred name and color
3. Configure the email subject, heading, and content
4. Use shortcodes to personalize the emails
5. Preview the email before saving
6. The new status will appear in the WooCommerce order status dropdown
7. When you change an order to that status, the customer receives your custom email

This plugin enhances the WooCommerce order management experience by providing flexibility to match your specific business workflow while keeping customers informed at every stage.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/as-custom-order-status` directory, or install the plugin through the WordPress plugins screen directly
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Navigate to WooCommerce > Custom Order Statuses to start creating your custom statuses
4. Create your first custom status by filling in the form
5. Use the preview feature to see how your emails will look
6. Save your custom status and start using it in your orders

== Frequently Asked Questions ==

= Can I edit existing custom statuses? =

Yes, you can edit any existing custom status by clicking the "Edit" link next to the status in the list.

= Can I preview emails before sending? =

Yes, there's a "Preview Email" button on both the add and edit forms that shows you exactly how your email will look.

= What shortcodes can I use in emails? =

You can use the following shortcodes:
* {order_number} - Displays the order number
* {customer_name} - Displays the customer's first name
* {order_date} - Displays the order date

= Can I delete custom statuses? =

Yes, you can delete any custom status by clicking the "Delete" link next to the status in the list.

= Will this work with my existing WooCommerce setup? =

Yes, the plugin is designed to work seamlessly with any standard WooCommerce installation.
